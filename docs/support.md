# TailyDose Support

Last updated: April 22, 2026

TailyDose helps you track pet medications, reminder schedules, dose history, and vet records.

## Support

If you need help with the app, purchase restore, reminder issues, or export behavior, open an issue in this repository:

[GitHub Issues](https://github.com/casstao1/TailyDose/issues)

## Common Topics

- Restoring a Pro purchase
- Reminder notifications not appearing
- Updating pet or medication details
- Exporting medication summaries for a veterinarian

## Response

Support is provided on a best-effort basis through the repository and future support channels listed there.
