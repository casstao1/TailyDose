Timeline Card — 7 Type-Scale Options

Seven directions for collapsing the current 3-size hierarchy. Each uses a constrained scale with weight + color carrying most of the hierarchy. Every option shows the card with and without a description so you can see how it flexes.

Option 1

Time hero, everything else matched

Time stays the visual anchor (26pt). Everything else — pet name, med name, description — at one unified size (15pt). Hierarchy comes from weight and color only.
